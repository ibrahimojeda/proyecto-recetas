Alimentos Quantum — Corrección Automatizada de Registros vacíos o con puntos (`.`)

¿Cómo usar esta herramienta?

  1. Exporta tu reporte PDF a un archivo Excel (.xlsx) o CSV.
  2. Asegúrate de que las columnas mantengan sus nombres originales (Producto, Ingrediente, Lote ingrediente, Lote salida, etc.).
  3. Arrastra el archivo al recuadro de abajo o haz clic para seleccionarlo.
  4. El sistema buscará los lotes correctos para el mismo bloque de producción y rellenará las celdas vacías o que contengan solo un punto.
  5. Descarga el archivo corregido de inmediato de forma segura.
